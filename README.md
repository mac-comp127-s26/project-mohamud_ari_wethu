# Target Practice

We want to create a 1-player shooting game where the player is shooting a ball at targets. The goal of the game is to shoot all the targets; for each miss, you lose 1 life. The player starts out with 3 lives. If they lose all 3 lives, they lose the game; if they shoot all the targets before they run out of lives, they win. We could potentially add obstacles and maybe even bouncing to the game, as well as making the targets move. The targets could be in different shapes, sizes, and colors.
